# Изображения для слайдера

В этой папке должны находиться 4 изображения для слайдера на главной странице:

- `slide1.jpg` - Первое изображение (Образование)
- `slide2.jpg` - Второе изображение (Курсы)
- `slide3.jpg` - Третье изображение (Профессионализм)
- `slide4.jpg` - Четвертое изображение (Успех)

## Требования к изображениям

- Формат: JPG, PNG или WebP
- Рекомендуемый размер: 1200×600px или больше
- Соотношение сторон: 2:1 (ширина:высота)
- Оптимизированы для веб (сжатие)

## Заглушки

Если изображения отсутствуют, слайдер автоматически отобразит цветные заглушки с текстом.

## Размещение

Изображения должны быть размещены в папке `frontend/images/` относительно корня проекта.
